# 链表项目（C语言实现）
这是一个用 C 语言从头实现的**单向链表库**。该项目旨在通过完整的数据结构实践，巩固C语言的指针操作、内存管理与模块化编程能力。

## 功能列表
- [x] **创建节点**：使用 `createNode()` 动态分配内存并初始化节点。
- [x] **头部插入**：实现 `insertAtHead()`，支持修改头指针。
- [x] **尾部插入**：实现 `insertAtEnd()`，遍历至链表末尾添加新节点。
- [x] **打印链表**：实现 `printList()`，格式化输出链表所有元素。
- [x] **获取长度**：实现 `getLength()`，遍历链表统计节点个数。
- [x] **按值查找**：实现 `searchNode()`，返回第一个匹配节点的指针。
- [x] **按值删除**：实现 `deleteNode()`，处理头节点删除及内存释放。
- [x] **反转链表**：实现 `reverseList()`，利用三指针迭代法改变节点指向。

### 环境与运行
- **开发环境**：Visual Studio 2022
- **运行方式**：下载项目，直接打开 `LinkedListProject.sln` 解决方案文件，按 `F5` 键即可编译运行。

### 预期输出
程序运行后，控制台将依次展示以下测试用例的结果，以验证各项功能的正确性：
1.  **尾插法**：依次插入 1, 2, 3, 4, 5
2.  **头插法**：依次插入 6, 7, 8
3.  **获取长度**：输出当前链表长度
4.  **按值查找**：查找指定元素
5.  **按值删除**：删除指定节点
6.  **链表反转**：反转链表并打印结果
