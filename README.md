# 链表项目（C语言实现）
这是一个用 C 语言实现的单向链表项目，包含完整的增删改查、反转等功能。

## 功能列表
- [1] 创建节点
- [2] 头插法
- [3] 尾插法
- [4] 打印链表
- [5] 获取链表长度
- [6] 按值查找
- [7] 按值删除
- [8] 反转链表

## 如何运行
用 Visual Studio 2022 打开 `LinkedListProject.sln`，直接按 F5 运行。

## 测试结果
运行后控制台会依次输出：
- 尾插 1,2,3,4,5
- 头插 6,7,8
- 链表长度
- 按值查找
- 删除节点
- 反转链表

## 项目结构
linkedlist.h - 头文件（函数声明）
linkedlist.c - 链表功能实现
main.c - 测试代码
